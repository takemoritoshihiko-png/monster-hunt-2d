# MONSTER HUNT 2D

見下ろし型2Dアクションゲーム。大型モンスターを狩猟し、素材を集めて武器・防具をクラフトするブラウザゲームです。

## 遊び方

| キー | アクション |
|------|-----------|
| WASD | 移動 |
| Z | 攻撃（近接 or 弓） |
| Q | 武器切り替え |
| I | インベントリ画面 |
| C | クラフト画面 |
| R/Enter | ロビーに戻る（結果画面時） |
| Click | クエスト選択（ロビー画面） |

## ローカル実行

`index.html` をブラウザで開くだけで動作します。

## Phase 1

- プレイヤー移動・近接攻撃
- 大型モンスター「Forest Drake」（HP500）
- 追跡AI・HPバー・ゲームオーバー

## Phase 2

- 素材ドロップシステム（Drake Scale / Fang / Core）
- インベントリ画面（I キー）
- クラフト画面（C キー）：Iron Sword / Hunter Bow / Drake Armor
- 弓（Hunter Bow）：矢が飛翔してダメージ
- 武器サイクル切り替え（Q キー）
- 防具による被ダメージ軽減
- モンスター15秒リスポーン

## Phase 3

- クエスト選択ロビー画面（3種のクエスト）
- Forest Drake討伐（難易度★☆☆）
- Drake 2体同時討伐（難易度★★☆）
- Giant Drake討伐（HP1500・突進攻撃・難易度★★★）
- 複数モンスター同時管理
- Giant Drake：HP50%以下で突進攻撃（予告エフェクト付き）
- 防具ダメージ倍率方式（Drake Armor: x0.7）
- クエスト結果画面（報酬一覧・5秒自動帰還）
- Web Audio API 効果音（ヒット・取得・クリア・失敗・突進予告）
