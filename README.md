# MONSTER HUNT 2D

見下ろし型2Dアクションゲーム。大型モンスターを狩猟するブラウザゲームです。

## 遊び方

| キー | アクション |
|------|-----------|
| WASD | 移動 |
| Z | 近接攻撃（剣） |
| R | リスタート（ゲームオーバー/クリア時） |

## ローカル実行

`index.html` をブラウザで開くだけで動作します。

## Phase 1 実装内容

- プレイヤー移動・攻撃
- 大型モンスター「Forest Drake」（HP500）
- 追跡AI（距離300px以内で追跡）
- HPバー表示（プレイヤー/モンスター）
- ゲームオーバー / 勝利画面
